Der IndexModel Typ exponiert die folgenden Member.

Methoden
  NameBeschreibung
OnGet
Dieser Event Handler wird aufgerufen, wenn die Weboberfläche angefordert wird.

Er konfiguriert gegebenenfalls die Tabelle gemäß der Suchanfrage.

OnPost
Dieser Event Handler wird aufgerufen, sobald das "POST"-Event auslöst wird (hier durch Betätigung des "Suchen"-Buttons).

Er lädt die Weboberfläche neu, sodass der Suchanfrage Rechnung getragen werden kann.
