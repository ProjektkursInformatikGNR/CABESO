Konvertiert ein NullableT-Objekt in einen Geldbetrag anhand der Methode GetDisplayFormat(Decimal).
GetDisplayFormat(DateTime, Boolean)
Konvertiert ein DateTime-Objekt in ein lesbares Format ("dd.MM.yyyy HH:mm[:ss]").
GetDisplayName
Gibt die anzuzeigende Bezeichnung einer gegebenen Rolle zurück.

Gibt die ihm zugehörige Schulklasse eines Benutzers zurück.
GetFormsSelect
Gibt eine Datenstruktur von Schulklassen zurück, zwischen denen der Benutzer typischerweise wählen kann.
GetHtmlDateFormat
Konvertiert ein DateTime-Objekt in das von HTML geforderte Tagesformat ("yyyy-MM-dd").
GetHtmlFormat
Konvertiert ein DateTime-Objekt in das von HTML geforderte Uhrzeitformat ("yyyy-MM-ddTHH:mm").
GetIdentityUser
Gibt anhand des ClaimsPrincipal den zugeordneten Benutzer zurück.
GetName
Gibt den Namen eines gegebenen Benutzers zurück.
GetRole
Gibt ggf. die Schüler*innen- bzw. Lehrer*innen-Rolle des Benutzers zurück.
GetSqlFormat
Konvertiert ein DateTime-Objekt in das von SQL geforderte Format ("yyyyMMdd hh:mm:ss tt").
GetUserById
Gibt anhand der ID den zugeordneten Benutzer zurück.
IsAdmin
Gibt an, ob der Benutzer ein Administrator ist.
IsEmployee
Gibt an, ob der Benutzer ein Mitarbeiter bzw. eine Mitarbeiterin ist.
MigrateDatabase
Bindet eine Instanz des ApplicationDbContext in den Migrationsvorgang ein.
SearchT
Durchsucht eine Datenstruktur nach einem Ausdruck und gibt zutreffende Elemente zurück.
SetFormId
Legt die Schulklassenzugehörigkeit eines Benutzers anhand der ID der Schulklasse fest.
