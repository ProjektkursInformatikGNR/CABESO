Methoden
  NameBeschreibung
OnGet
Dieser Event Handler wird aufgerufen, wenn die Weboberfläche angefordert wird.

Er initialisiert die zu bearbeitende Bestellung CurrentOrder anhand der ID.

OnPost
Dieser Event Handler wird aufgerufen, sobald das "POST"-Event auslöst wird (hier durch Betätigung des "Bearbeiten"-Buttons).

Er bearbeitet auf Grundlage der eingegebenen Informationen die gegebenene Bestellung.
