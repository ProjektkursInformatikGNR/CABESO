CABESO.Areas.Orders.Pages Namensraum

Das Namespace [CABESO.Areas.Orders.Pages] beinhaltet Features zur Aufgabe von Bestellungen durch den Benutzer und die Einsicht und Verwaltung bereits getätigter Bestellungen.
