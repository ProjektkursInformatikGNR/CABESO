Methoden
  NameBeschreibung
OnGetAsync
Dieser Event Handler wird aufgerufen, wenn die Weboberfläche angefordert wird.

Er überprüft, ob aktuell ein gültiger Benutzer angemeldet ist.

OnPostAsync
Dieser Event Handler wird aufgerufen, sobald das "POST"-Event auslöst wird (hier durch Betätigung des "Passwort ändern"-Buttons).

Er ändert das Passwort des aktuellen Benutzers und gibt eine Rückmeldung (entweder Statusmeldung oder Fehlermeldung).
