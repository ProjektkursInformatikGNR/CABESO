OnGet
Dieser Event Handler wird aufgerufen, wenn die Weboberfläche angefordert wird.

Er initialisiert ggf. die Eigenschaft Code und zeigt andernfalls eine Zugriffsablehnung an.

OnPostAsync
Dieser Event Handler wird aufgerufen, sobald das "POST"-Event auslöst wird (hier durch Betätigung des "Passwort zurücksetzen"-Buttons).

Er verifiziert den Passwortzurücksetzungscode und ersetzt im Erfolgsfall das Benutzerpasswort durch das eingegebene.
