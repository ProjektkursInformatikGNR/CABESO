Methoden

FromInput
Konvertiert die Preiseingabe vom String-Format ins Decimal-Format.
OnGet
Dieser Event Handler wird aufgerufen, wenn die Weboberfläche angefordert wird.

Er initialisiert das zu bearbeitende Produkt CurrentProduct anhand der ID.

OnPost
Dieser Event Handler wird aufgerufen, sobald das "POST"-Event auslöst wird (hier durch Betätigung des "Bearbeiten"-Buttons).

Er bearbeitet auf Grundlage der eingegebenen Informationen das gegebenene Produkt.

ToInput
Konvertiert den Preisbetrag vom Decimal-Format ins HTML-Format als String.
