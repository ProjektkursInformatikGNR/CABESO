Methoden
  NameBeschreibung
OnGet
Dieser Event Handler wird aufgerufen, wenn die Weboberfläche angefordert wird.

Er verifiziert die Benutzer-ID und initialisiert im Erfolgsfall die URL-Daten.

OnPostAsync
Dieser Event Handler wird aufgerufen, sobald das "POST"-Event auslöst wird (hier durch Betätigung des "Passwort einrichten"-Buttons).

Er verifiziert den Passworterstellungscode und legt im Erfolgsfall das neue Passwort fest.
